# 自闭症友好性增强建议

## 感官调节功能

- **音量控制** - 独立的音效/音乐音量滑块
- **视觉强度调节** - 泡泡闪烁/动画强度可调
- **色彩模式** - 高对比度/柔和色彩模式切换
- **减少突然变化** - 渐进式难度提升，避免突然的速度/颜色变化

## 可预测性增强

- **倒计时显示** - 清晰的剩余时间指示
- **进度条** - 视觉化的游戏进度
- **预告机制** - 提前告知即将发生的变化（如速度提升）
- **固定模式** - 可选择固定的泡泡出现模式，减少随机性

## 成功体验优化

- **渐进式挑战** - 从简单开始，逐步增加复杂度
- **多样化成功** - 不只是"戳中"，还有"接近"、"尝试"等鼓励
- **个性化目标** - 根据用户能力调整目标
- **成就系统** - 小步骤的成就感积累

## 社交互动支持

- **家长/治疗师模式** - 可以观察和记录进展
- **协作模式** - 双人协作戳泡泡
- **分享功能** - 生成的音乐可以分享给家人

## 数据追踪

- **运动数据** - 手臂活动范围、协调性指标
- **进步记录** - 长期的能力发展追踪
- **个性化报告** - 为治疗师提供有用的数据
