# 自闭症友好泡泡游戏 - 使用指南

## 🎯 游戏目标
通过在空间中移动手臂来戳破泡泡，锻炼手眼协调能力，并在游戏结束时获得根据你的动作生成的独特音乐。

## 🎛️ 感官调节功能

### 访问感官设置
1. 点击游戏界面上的 "🎛️ 感官设置" 按钮
2. 设置面板将出现，包含以下选项：

### 音效控制
- **音效音量**: 0-100% 可调节
- 建议从较低音量开始，逐渐调整到舒适水平

### 视觉调节
- **动画强度**: 5个级别（很慢 → 很快）
  - 很慢/慢：适合对快速变化敏感的用户
  - 中等：标准设置
  - 快/很快：适合喜欢快节奏的用户

- **色彩模式**:
  - 标准：柔和的彩色泡泡
  - 高对比度：黑白高对比，适合视觉处理困难的用户
  - 柔和：更温暖、更柔和的色调

### 可预测性设置
- **可预测模式**: 启用后泡泡将按固定模式出现
  - 减少随机性带来的焦虑
  - 帮助建立预期和安全感
  - 右上角会显示 "🔄 可预测模式已启用" 提示

## 🎮 游戏控制

### 速度调节
- **🐌 慢速**: 适合初学者或需要更多时间反应的用户
- **🚶 正常**: 标准游戏速度
- **🏃 快速**: 挑战模式，适合熟练用户

### 输入方式
1. **鼠标模式**: 移动鼠标指针戳泡泡
2. **手势模式**: 点击 "📷 摄像头" 按钮启用
   - 伸出食指即可控制
   - 需要摄像头权限

### 暂停功能
- 点击 "⏸️ 暂停" 按钮或按空格键
- 游戏会完全停止，给你时间休息

## 📊 进度显示

### 时间指示器
- **剩余时间**: 显示当前回合剩余时间
- **进度条**: 视觉化显示游戏进度

### 状态信息
- **输入方式**: 显示当前使用鼠标还是手势
- **泡泡数**: 显示屏幕上的泡泡数量

## 🏆 成就系统

游戏会自动识别你的进步并给予鼓励：
- **连续成功**: 连续戳中5个泡泡
- **里程碑**: 总共戳中10个、50个泡泡等
- 成就会以友好的弹窗形式显示

## 🎵 AI音乐生成

### 工作原理
- 游戏记录你戳泡泡时产生的音符
- 回合结束时，AI会基于这些音符创作一段旋律
- 音乐会自动播放，并可下载为MIDI文件

### 音乐特点
- 每次游戏都会产生独特的音乐
- 音乐反映了你的游戏节奏和模式
- 可以作为个人成就的纪念

## 🧠 治疗价值

### 感觉统合训练
- **本体觉**: 通过手臂运动感知身体位置
- **视觉追踪**: 跟踪移动的泡泡
- **手眼协调**: 精确控制手部动作

### 认知技能
- **注意力集中**: 专注于目标泡泡
- **预测能力**: 预判泡泡移动轨迹
- **反应速度**: 快速响应视觉刺激

### 情绪调节
- **成就感**: 通过成功戳泡泡获得积极反馈
- **创造性**: 通过AI音乐生成表达个性
- **自我控制**: 通过暂停功能管理感官输入

## 💡 使用建议

### 初次使用
1. 从慢速模式开始
2. 调低音量到舒适水平
3. 启用可预测模式减少焦虑
4. 使用鼠标模式（更容易控制）

### 进阶使用
1. 逐渐提高游戏速度
2. 尝试手势控制增加身体参与
3. 关闭可预测模式增加挑战
4. 调整动画强度找到最佳设置

### 家长/治疗师指导
- 观察用户的反应，及时调整设置
- 鼓励尝试不同的输入方式
- 记录进步情况（游戏会自动追踪数据）
- 将生成的音乐作为成就分享

## 🔧 故障排除

### 摄像头问题
- 确保使用HTTPS访问（已配置）
- 检查浏览器摄像头权限
- 尝试刷新页面重新授权

### 音频问题
- 首次使用需要点击页面激活音频
- 检查系统音量设置
- 尝试调整游戏内音量滑块

### 性能问题
- 降低动画强度
- 关闭其他占用资源的程序
- 使用较新的浏览器版本

## 📈 数据追踪

游戏会自动记录：
- 运动轨迹和平滑度
- 成功率和反应时间
- 游戏偏好设置
- 长期进步趋势

这些数据可以帮助治疗师了解用户的发展情况。
